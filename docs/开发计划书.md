## 一、项目背景与目标

### 1.1 需求来源

根据参考视频（心理疗愈类学习 App），提炼核心用户路径：**在旅程地图中浏览课程进度 → 进入课程学习长文内容 → 完成反馈 → 解锁下一节点**。本 Demo 聚焦该主流程，验证全栈架构设计、前后端协作与国际化能力。

### 1.2 项目目标

构建一个**可运行、可打包、可扩展**的 Demo，满足以下目标：

1. 前后端分离，接口契约清晰，便于多人并行开发
2. 数据库支持章节、课程、进度、多语言文案
3. 前端还原参考视频的核心交互与视觉风格
4. 课程正文超过 5000 字，支持富文本排版
5. 前后端均实现国际化（`zh-CN` / `en-US`）

### 1.3 技术选型

| 层级 | 技术 | 选型理由 |
|------|------|----------|
| 前端 | React Native (Expo SDK 52) + TypeScript + expo-router | 符合岗位要求，一套代码打包 Web / Android |
| 后端 | Node.js + Express + Prisma | 与前端同语言栈，Prisma 简化 ORM 与迁移 |
| 数据库 | SQLite | Demo 轻量部署，可平滑迁移 PostgreSQL |
| 国际化 | i18next（前端）+ Translation 表（后端） | 前后端文案分离，符合内容国际化要求 |
| 部署 | Docker（后端）+ expo export / EAS Build | 覆盖服务端部署与移动端打包 |

│   └── 录屏脚本.md

## 二、功能拆解与验收标准

### 功能 1：底部 Tab 导航

| 项 | 说明 |
|----|------|
| **目标** | 实现 5 个 Tab：今日 / 旅程 / 探索 / 陪伴 / 个人资料，默认进入「旅程」 |
| **验收方式** | 运行后底部导航可切换；非核心 Tab 显示占位文案，不影响主流程 |

### 功能 2：课程地图（旅程页）

| 项 | 说明 |
|----|------|
| **目标** | 生成可上下滚动的课程地图，展示章节信息、节点状态（可学 / 已完成 / 锁定）、星星积分 |
| **验收方式** | 运行后可见 Chapter 标签、蜿蜒路径布局；锁定节点呈灰色锁态，可学节点呈金色高亮 |

### 功能 3：章节切换与详情

| 项 | 说明 |
|----|------|
| **目标** | 点击章节头部展开 Bottom Sheet，展示章节介绍与探索要点，支持切换 Chapter 1 / 2 |
| **验收方式** | Sheet 弹出流畅；bullet 列表与参考视频一致；可切换至 Chapter 2 查看锁定路径 |

### 功能 4：节点交互弹窗

| 项 | 说明 |
|----|------|
| **目标** | 点击可学节点弹出课程卡片，展示标题、类型（Reading / Practice）、操作按钮 |
| **验收方式** | 锁定节点点击无响应；可学节点弹出 Start；已完成节点显示 Continue |

### 功能 5：课程学习全流程

| 项 | 说明 |
|----|------|
| **目标** | 封面页（序号 + 时长 + 标题）→ 正文阅读（进度条 + 字号调节）→ 完成页（表情评分） |
| **验收方式** | 正文可流畅滚动；顶部绿色进度条随阅读推进；完成后返回地图，节点状态更新 |

### 功能 6：进度解锁与积分

| 项 | 说明 |
|----|------|
| **目标** | 完成课程后获得星星奖励，自动解锁同章节下一节课 |
| **验收方式** | 完成第 1 节后星星 +1，第 2 节由锁定变为可学；第 1 节节点变为已完成态 |

### 功能 7：国际化

| 项 | 说明 |
|----|------|
| **目标** | 支持 `zh-CN` / `en-US`；前端 UI 文案与后端课程数据均按语言返回 |
| **验收方式** | 在 Profile 切换语言后，Tab 标签、按钮、课程标题、正文内容同步切换 |

### 功能 8：长文富文本排版

| 项 | 说明 |
|----|------|
| **目标** | 第一节课程正文超过 5000 字，包含标题、段落、有序 / 无序列表 |
| **验收方式** | API 返回 `contentLength > 5000`；前端正确渲染 HTML 层次结构 |

## 三、数据库表结构设计

采用 **主表 + Translation 表** 模式，将结构与多语言文案解耦，便于后续扩展更多语言。

```prisma
model Chapter {
  id           Int                  @id @default(autoincrement())
  sortOrder    Int
  slug         String               @unique
  translations ChapterTranslation[]
  lessons      Lesson[]
}

model ChapterTranslation {
  id          Int     @id @default(autoincrement())
  chapterId   Int
  locale      String              // "en-US" | "zh-CN"
  title       String              // e.g. "Introduction"
  label       String              // e.g. "CHAPTER 1"
  description String?
  bullets     String              // JSON: ["...", "..."]
  @@unique([chapterId, locale])
}

model Lesson {
  id          Int    @id @default(autoincrement())
  chapterId   Int
  sortOrder   Int
  type        String               // "reading" | "practice"
  durationMin Int    @default(3)
  starReward  Int    @default(1)
  translations LessonTranslation[]
  progress     UserLessonProgress[]
}

model LessonTranslation {
  id       Int    @id @default(autoincrement())
  lessonId Int
  locale   String
  title    String
  subtitle String?
  content  String                 // HTML, 首课 >5000 字
  @@unique([lessonId, locale])
}

model UserProfile {
  id       Int    @id @default(1)
  stars    Int    @default(0)
  locale   String @default("en-US")
  progress UserLessonProgress[]
}

model UserLessonProgress {
  id          Int       @id @default(autoincrement())
  userId      Int       @default(1)
  lessonId    Int
  status      String                 // locked | available | completed
  rating      Int?
  completedAt DateTime?
  @@unique([userId, lessonId])
}
```

### 种子数据规划

| 章节 | 课程数 | 说明 |
|------|--------|------|
| Chapter 1 · Introduction | 4 | 第 1 节默认 available，含 5000+ 字长文 |
| Chapter 2 · Thinking | 3 | 全部 locked，用于展示章节切换 |

### 解锁规则

1. 初始状态：Chapter 1 第 1 节 `available`，其余 `locked`
2. 完成当前课 → 同章节 `sortOrder + 1` 的课程变为 `available`
3. 章节最后一课完成 → 下一章节第 1 课变为 `available`
4. 每完成一课，`stars += starReward`（默认 1）

## 四、前端设计思路

### 4.1 架构分层

```
┌─────────────────────────────────────┐
│  Screens (expo-router)              │
│  (tabs) / lesson/[id]/*            │
├─────────────────────────────────────┤
│  Components (journey / lesson)      │
├─────────────────────────────────────┤
│  Services (api.ts) + Context (i18n) │
├─────────────────────────────────────┤
│  Constants (theme, assets)          │
└─────────────────────────────────────┘
```

### 4.2 目录结构

```
spark-demo-app/
├── app/
│   ├── (tabs)/                 # 底部 Tab 页面
│   │   ├── journey.tsx         # 核心：旅程地图
│   │   └── profile.tsx         # 语言切换
│   ├── lesson/[id]/
│   │   ├── intro.tsx           # 课程封面
│   │   ├── read.tsx            # 正文阅读
│   │   └── complete.tsx        # 完成评分
│   └── _layout.tsx
├── components/
│   ├── journey/                # 节点、弹窗、章节 Sheet
│   └── lesson/                   # 顶栏、评分组件
├── services/api.ts
├── context/AppContext.tsx      # locale + fontScale
├── i18n/index.ts
└── assets/images/              # 参考资源包 light/
```

### 4.3 关键 UI 实现方案

| 模块 | 实现方案 | 说明 |
|------|----------|------|
| 蜿蜒路径 | `ScrollView` + 节点 `alignSelf` 交替偏移 | 模拟 Duolingo 风格曲线路径 |
| 节点状态 | 组合 PNG 素材（base / inner / outer / icon） | 使用官方 light 资源包 |
| 章节 Sheet | `Modal` + 底部滑入动画 | 轻量实现，无额外依赖 |
| 富文本渲染 | `react-native-render-html` | 解析后端 HTML 正文 |
| 阅读进度 | `onScroll` 计算 `offset / (contentHeight - viewportHeight)` | 驱动顶部绿色进度条 |
| 字号调节 | Context 管理 `fontScale`（sm / md / lg） | Aa 按钮循环切换 |
| 国际化 | `i18next` + `expo-localization` | UI 文案本地化管理 |

### 4.4 状态与数据流

1. 进入旅程页 → `GET /journey` 拉取章节 + 进度
2. 点击节点 → 本地判断 `status !== 'locked'` → 导航至 `/lesson/:id/intro`
3. 阅读完成 → `POST /lessons/:id/complete` → `router.replace` 回旅程页
4. 页面 `useFocusEffect` 自动刷新，展示最新进度

## 五、后端设计思路

### 5.1 架构分层

```
Request → locale middleware → Router → Service → Prisma → SQLite
```

### 5.2 目录结构

```
spark-demo-api/
├── src/
│   ├── app.ts                  # Express 入口
│   ├── routes/index.ts         # 路由注册
│   ├── services/journeyService.ts  # 业务逻辑
│   └── middleware/locale.ts    # 语言解析
├── prisma/
│   ├── schema.prisma
│   └── seed.ts                 # 种子数据（含 5000+ 字正文）
└── Dockerfile
```

### 5.3 核心业务逻辑

| 方法 | 职责 |
|------|------|
| `getJourney(locale)` | 聚合章节、课程、用户进度，计算节点 status |
| `getLessonDetail(id, locale)` | 返回课程详情与 HTML 正文 |
| `completeLesson(id, rating)` | 更新进度、累加星星、触发解锁 |
| `pickTranslation(items, locale)` | 按 locale 取文案，fallback `en-US` |

### 5.4 语言解析策略

优先级：`?locale=` 查询参数 > `Accept-Language` Header > 默认 `en-US`

    "status": "completed",
