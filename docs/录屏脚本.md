# Spark Demo 录屏脚本

> 重庆亦之趣科技 · 程序开发二测 · 建议时长 4–6 分钟

## 录制前准备

1. **重置数据库**（必须，两章各 7 课；过期库会导致节点数量异常）：
   ```bash
   cd spark-demo-api
   PRISMA_USER_CONSENT_FOR_DANGEROUS_AI_ACTION='可以' npm run db:reset
   ```
   终端应出现：`Chapter 1: 7 lessons | Chapter 2: 7 lessons | Total: 14`
2. 启动后端：`npm run dev`（保持终端可见；启动日志也会打印各章课时）
3. 启动前端 Web：`cd spark-demo-app && npm run web`
4. 浏览器打开 `http://localhost:8081`（或 Expo 提示的地址）
5. 录屏工具：QuickTime / OBS，分辨率 1080×1920 或手机竖屏比例
6. 确认默认语言为**中文**（若曾切英文，先到个人资料切回中文）

---

## 分镜脚本

### 第一幕：项目介绍（约 30 秒）

**画面**：展示项目目录结构（VS Code 或终端 `tree`）

**旁白参考**：
> 这是根据参考视频实现的 Spark 疗愈旅程 Demo。前后端分离，前端 Expo + React Native，后端 Express + Prisma + SQLite，支持中英文国际化，默认中文。

**操作**：
- 短暂展示 `docs/开发计划书.md`
- 展示 `spark-demo-api` 和 `spark-demo-app` 两个目录

---

### 第二幕：启动服务（约 30 秒）

**画面**：两个终端窗口

**旁白参考**：
> 后端运行在 3000 端口，提供旅程地图和课程内容的 REST API。种子数据两章各 7 个课程、共 14 个节点。第一节正文超过 5000 字，带 HTML 排版。

**操作**：
```bash
# 终端 1
cd spark-demo-api && npm run dev

# 终端 2
cd spark-demo-app && npm run web
```

可选：
```bash
curl 'http://localhost:3000/api/v1/journey?locale=zh-CN'
```
指出 JSON 里两章 `lessons` 数组长度均为 7，第 2 章 `sections` 含 `beforeLessonOrder: 1`。

---

### 第三幕：旅程地图（约 60 秒）

**画面**：App 旅程 Tab

**旁白参考**：
> 这是核心页面——课程地图。顶部显示章节信息和星星积分，中间是 S 形蜿蜒学习路径——节点用连续水平偏移排布，不是简单的左中右三档。金色节点是当前可学，右上角有一组大小不一的星星在小范围内旋转闪烁；灰色带锁的是未解锁课程。

**操作**：
1. 展示底部 5 个 Tab，停留在「旅程」
2. 指出右上角星星为 0；当前金色节点的**星星簇动画**
3. 缓慢下滑，展示第 1 章连续 **7 个**节点（中间**没有**分隔线）
4. 继续下滑：接近第 2 章时，顶部标题切到「第 2 章 / 思维」；路径上出现 **「思维」分隔线**，其后又是 7 个锁定节点
5. 点击章节头部，展开 Bottom Sheet，展示当前章节介绍与 bullet 列表
6. 关闭 Sheet；可用右下角定位按钮滚回当前章节

**注意口播**：
> 顶部标题随滚动锚点切换；若标题还停在第 1 章，说明尚未滚到第 2 章起点。不要把「第 2 章的前几个节点」误当成第 1 章。

---

### 第四幕：课程学习全流程（约 100 秒）

**画面**：从节点到完成

**旁白参考**：
> 点击当前可学习的节点，弹出课程卡片。封面页之后进入长文阅读：内容按段落分批展示。点击 Continue 时，加载动画出现在已显示正文的最下方，然后下一段从上而下展开，而不是整页硬翻。

**操作**：
1. 点击金色节点 → 弹窗显示课程标题 → 点 **开始**
2. 封面页：展示序号、时长、标题 → 点 **开始课程**
3. 阅读页：展示首段正文 → 点 **Aa** 切换字号
4. 点 **继续**：指出正文底部的加载菊花 → 新段落向下 reveal 展开
5. 重复 Continue 若干次，展示进度条随段落推进
6. 最后一段点 **完成本次课程**
7. 完成页：选择评分表情 → 点 **继续**
8. 回到地图：第一个节点变已完成，第二个节点解锁（带星星簇），星星变为 1

---

### 第五幕：国际化（约 30 秒）

**画面**：Profile Tab → 旅程 / 课程

**旁白参考**：
> 前端 UI 和后端课程数据都支持国际化。切换到英文后，Tab、章节头、按钮和课程标题、正文都会变成英文，不会中英混排。

**操作**：
1. 进入「个人资料」Tab
2. 点击 English / 英文
3. 回到旅程 Tab：确认 Chapter / Journey 等英文文案，路径与章节头同步切换
4. 可选：再次进入第一节，确认标题与正文为英文
5. 录屏结束前可切回中文

---

**画面**：终端执行打包命令

## 录制注意事项

- 录屏前必须 `db:reset`，确保：星星 0、第一节可学、两章各 7 课
- 操作节奏放慢；下滑切章时等顶部标题变化后再说话
- 演示 Continue 时镜头对准**正文底部菊花**，体现「加载在文末」
- 终端字体调大，方便阅卷人看清命令
- 若 Web 端动画略弱于真机，可口播「星星簇与路径布局在移动端更接近参考视频」
- 最终导出 MP4，文件名建议：`spark-demo-录屏.mp4`
