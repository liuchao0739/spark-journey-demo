# Spark Demo 录屏脚本

> 重庆亦之趣科技 · 程序开发二测 · 建议时长 4–6 分钟

## 录制前准备

1. 重置数据库到初始状态：
   ```bash
   cd spark-demo-api && npm run db:reset
   ```
2. 启动后端：`npm run dev`（保持终端可见）
3. 启动前端 Web：`cd spark-demo-app && npm run web`
4. 浏览器打开 `http://localhost:8081`（或 Expo 提示的地址）
5. 录屏工具：QuickTime / OBS，分辨率 1080×1920 或手机竖屏比例

---

## 分镜脚本

### 第一幕：项目介绍（约 30 秒）

**画面**：展示项目目录结构（VS Code 或终端 `tree`）

**旁白参考**：
> 这是根据参考视频实现的 Spark 疗愈旅程 Demo。前后端分离，前端 Expo + React Native，后端 Express + Prisma + SQLite，支持中英文国际化。

**操作**：
- 短暂展示 `docs/开发计划书.md`
- 展示 `spark-demo-api` 和 `spark-demo-app` 两个目录

---

### 第二幕：启动服务（约 30 秒）

**画面**：两个终端窗口

**旁白参考**：
> 后端运行在 3000 端口，提供旅程地图和课程内容的 REST API。前端通过接口拉取数据，课程正文超过 5000 字，带 HTML 排版。

**操作**：
```bash
# 终端 1
cd spark-demo-api && npm run dev

# 终端 2
cd spark-demo-app && npm run web
```

可选：执行 `curl http://localhost:3000/api/v1/journey?locale=en-US` 展示 JSON 返回

---

### 第三幕：旅程地图（约 45 秒）

**画面**：App 旅程 Tab

**旁白参考**：
> 这是核心页面——课程地图。顶部显示章节信息和星星积分，中间是蜿蜒的学习路径。金色节点表示可学习或已完成，灰色带锁的是未解锁课程。

**操作**：
1. 展示底部 5 个 Tab，停留在「旅程」
2. 指出右上角星星为 0
3. 点击章节头部，展开 Bottom Sheet，展示 Chapter 1 介绍和 bullet 列表
4. 切换到 Chapter 2（Thinking），展示锁定状态
5. 关闭 Sheet，回到 Chapter 1

---

### 第四幕：课程学习全流程（约 90 秒）

**画面**：从节点到完成

**旁白参考**：
> 点击当前可学习的节点，弹出课程卡片。进入后先看封面页，再阅读长文内容。顶部进度条随滚动更新，右上角 Aa 可调节字号。

**操作**：
1. 点击金色节点 → 弹窗显示课程标题 → 点 **Start**
2. 封面页：展示序号 01、时长 3 mins、标题 → 点 **Start session**
3. 阅读页：向下滑动浏览正文（展示标题、列表等排版）→ 点 **Aa** 切换字号
4. 滑到底部 → 点 **Complete session**
5. 完成页：选择 5 星表情 → 点 **Continue**
6. 回到地图：第一个节点变已完成，第二个节点解锁，星星变为 1

---

### 第五幕：国际化（约 30 秒）

**画面**：Profile Tab

**旁白参考**：
> 前端 UI 和后端课程数据都支持国际化。切换到中文后，Tab 标签、按钮和课程标题、正文都会更新。

**操作**：
1. 进入「个人资料」Tab
2. 点击「中文」
3. 回到旅程 Tab，确认 Tab 标签变为中文
4. 再次进入第一节课程，确认标题和正文为中文

---

### 第六幕：打包演示（约 60 秒）

**画面**：终端执行打包命令

**旁白参考**：
> 项目支持 Web 静态导出和 Android APK 打包。这里演示 Web 导出。

**操作**：
```bash
cd spark-demo-app
npm run export:web
```
- 展示 `dist/` 目录生成
- 可选：用 `npx serve dist` 打开静态站点验证

**APK 补充说明（口播即可）**：
> Android APK 通过 EAS Build 打包：`eas build --platform android --profile preview`

---

### 收尾（约 15 秒）

**旁白参考**：
> 以上是完整 Demo。源码和开发计划书已开源在 GitHub，README 中有运行和部署说明。谢谢观看。

**画面**：展示 GitHub 仓库页面和 README

---

## 录制注意事项

- 录屏前执行 `db:reset`，确保星星为 0、第一节可学
- 操作节奏放慢，每个页面停留 2–3 秒
- 终端字体调大，方便阅卷人看清命令
- 若 Web 端布局与手机略有差异，可补充一句「移动端 APK 布局与参考视频一致」
- 最终导出 MP4，文件名建议：`spark-demo-录屏.mp4`
